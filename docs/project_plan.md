# Project Plan — Iris Flower Classification

## Goal
Classify iris flowers into 3 species: Setosa, Versicolor, Virginica

## Features I Will Build
1. Data loading and exploration (EDA)
2. Data cleaning and preprocessing
3. Feature engineering
4. Model training — KNN, Decision Tree, SVM
5. Model evaluation — accuracy, confusion matrix
6. Visualizations — pairplot, heatmap, confusion matrix
7. Model export using joblib/pickle
8. Final documentation and GitHub cleanup

## Timeline
- Week 1: Setup, research, EDA, cleaning
- Week 2: Model building
- Week 3: Evaluation & visualizations
- Week 4: Final touches & submission
